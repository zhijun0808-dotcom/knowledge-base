# 存款保险解决挤兑但引入道德风险

## 核心观点
**联邦存款保险公司（FDIC）1935年成立，彻底终结了破坏性银行挤兑**——存款人知道钱有保障，就不再恐慌性取款。但保险改变了激励：

1. **存款人不再监督银行**——钱有保险，不管银行怎么冒险都不担心
2. **银行敢冒更大险**——高风险=高收益，亏了由保险基金（纳税人）兜底
3. **监管必然跟上**——政府必须限制受保银行的风险行为，否则保险基金破产

**道德风险实证**：
- 州存款保险更早（为保护禁止分支机构的小银行），结果**禁止分支机构的州，大萧条时倒闭银行最多**——分支机构能跨地区分散风险，单一网点银行极脆弱
- 加拿大无存款保险（1935前），但全国分支机构网络，**大萧条期零银行倒闭**
- 美国1980s储蓄贷款危机、2008次贷危机：受保机构过度冒险，纳税人买单

**核心悖论**：存款保险解决了**流动性风险**（挤兑），却放大了**信用风险**（烂账）。

## 思考
这张卡揭示了**金融监管的根本困境**：
- 无保险 → 挤兑风险 → 系统性崩塌
- 有保险 → 道德风险 → 过度冒险 → 危机更大

最优解不是"有或无"，而是**激励对齐**：
- 共同保险（存款人分担损失）
- 风险定价保费（冒险银行多交费）
- 资本充足率硬约束
- 及时纠正行动（PCA）

加拿大模式启示：**全国分支机构分散风险，比存款保险更本质地解决脆弱性**。
