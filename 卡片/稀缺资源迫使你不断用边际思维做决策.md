# 稀缺资源迫使你不断用边际思维做决策

## 核心观点

经济学不是因为"有钱"而存在——恰恰相反，是因为**资源永远不够用**。

稀缺是你无法逃避的处境，边际思维是你应对这个处境的必备工具。

## 稀缺 → 选择 → 边际思维

```
稀缺（资源不够）
   ↓
被迫选择（不能什么都要）
   ↓
每个选择都是"多一个单位"的边际问题
   ↓
需要一套比较框架 → 边际思维（MB vs MC）
```

## 决策规则

| 条件 | 行动 |
|------|------|
| **MB > MC** | 继续做 / 多做一点 |
| **MB < MC** | 停止 / 少做一点 |
| **MB = MC** | 最优停止点 |

## 经典误区

| 误区 | 错误思维 | 正确思维 |
|------|---------|---------|
| **沉没成本谬误** | "都投了100万了，再投10万无所谓" | **只比下一步的投入产出** |
| **平均成本谬误** | "平均成本100元，卖90元亏本" | **看边际成本：再生产一件成本80元，卖100元就是赚** |

## 经典案例：飞机起飞前卖便宜票

| 视角 | 思考 |
|------|------|
| 平均成本 | "每个座位成本2000元，卖500元亏本！" |
| **边际思维** | "飞机反正要飞，多一个乘客的边际成本≈0，卖500元纯赚！" |

## 为什么是"不断"？

稀缺不会消失——你今天选了加班不健身，明天又要选一次。**每个单位资源都在逼你做选择。**

| 稀缺类型 | 对应的边际决策 |
|---------|--------------|
| 时间稀缺（24小时） | 多陪家人一小时 vs 多工作一小时 |
| 金钱稀缺（预算有限） | 多买一件衣服 vs 多存一千块 |
| 注意力稀缺（精力有限） | 多学Python vs 多学英语 |
| 资源稀缺（人手/设备有限） | 多招一个人 vs 多买一台机器 |

## 练习任务

今晚睡前想3个生活中的决策，用"MB > MC"分析一下：

- 要不要刷30分钟手机再睡？（MB=放松，MC=明天困、效率下降）
- 要不要考那个证？（MB=职业机会，MC=学习时间+金钱+机会成本）
- 要不要再投1万块广告？（MB=增量销售额×毛利，MC=1万广告费）

## 出处

- 曼昆《经济学原理》第1章——十大原理：人们面临权衡取舍
- 保罗·海恩《经济学的思维方式》第1章——稀缺与选择的关系

## 思考

以前总把"稀缺"和"边际思维"当两个独立的概念——一个讲资源不够，一个讲怎么做比较。现在才意识到：**稀缺就是"你逃不掉的选择"，边际思维就是"怎么选才不亏"的方法。** 没有稀缺，边际思维根本不需要存在。

## 检验

- ✅ 别人看完能用"MB > MC"判断一个具体决策（要不要加班、要不要考证、要不要投广告）
- ✅ 能说清楚稀缺和边际思维的关系：不是并列，是因果关系
- ✅ 能回答"资源无限还需要边际思维吗？"——不需要，因为没有选择压力了
