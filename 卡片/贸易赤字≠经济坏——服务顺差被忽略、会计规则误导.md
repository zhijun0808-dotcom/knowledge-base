# 贸易赤字≠经济坏——服务顺差被忽略、会计规则误导

## 核心观点
**"贸易赤字"只算货物、不算服务——美国服务顺差$6280亿（2012）被忽略，专利费/牌照费$1240亿。**

会计陷阱：
1. **只算物品不算服务**：美产服务>商品，进口商品多、出口服务多 → 货物逆差、服务顺差
2. **投资不算贸易**：日企赚美$→美建厂雇美工、买洛克菲勒中心 → 资金未离美、创造美就业
3. **投资算"债务"**：外资买美资产=美成"债务人" → 实则外资看好美经济、创造美就业
4. **收支平衡≠经济健康**：德常顺差但增速低/失业高；尼日利亚顺差却最穷；美92年衰退后罕见顺差

**核心洞察**：贸易赤字排第75大担忧——除非政客用关税/贬值反应。相对规模（占GDP%）比绝对额重要：美赤字5%GDP，<土耳其一半、马其顿四分一。
