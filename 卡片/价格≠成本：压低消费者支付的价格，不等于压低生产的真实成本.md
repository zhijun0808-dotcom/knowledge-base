# 价格≠成本：压低消费者支付的价格，不等于压低生产的真实成本

## 核心观点

**价格**是你为一件商品支付的数；**成本**是社会生产它真实耗费的数（资源、劳动、时间、知识）。压低消费者支付的价格（如医保限价、房租管制），**不等于压低生产的真实成本**——成本不会消失，只是被隐藏或转移。

## 为什么容易混淆

政客说"我要把药价/房价降下来"，听众默认"降价=成本变低=大家受益"。但：

| 你看到的 | 实际发生的 |
|---------|-----------|
| 医保限价 → 病人付得少 | 培养医生、建医院、研发药物的成本一分没少 → 医院亏损 → 服务萎缩 |
| 房租管制 → 租客付得少 | 盖房、修房、拿地的成本一分没少 → 房东不修不盖 → 房源变少变破 |
| 农产品保护价 → 农民卖得高 | 消费者多付的钱 + 纳税人补贴 = 真实成本，只是换人买单 |

## 成本去哪了

成本不消失，只是**转移**：

```
压低消费者支付的价格
  → 生产者利润被压缩
    → 生产者减产/降质/撤出
      → 成本转移到：排队、短缺、质量下降、纳税人
```

价格只是把"真实成本"用一个数字显示出来。**掐掉显示，成本还在，只是你看不见了。**

## 和价格管制的关系

这张卡是 `价格管制不会消除稀缺` 的补充说明：

- 那张卡说：管制不消除稀缺，只换配给方式
- 这张卡说：管制也不消除**成本**，只把成本从"明码标价"藏到"隐性代价"

两者同源——都是"信号被掐断，现实仍在"。

## 出处

- 托马斯·索维尔《经济学的思维方式》第 4 章——价格与成本
- 核心句："价格传达信息，不创造也不消灭成本"

## 思考

以前听到"政府把XX价格压下来"就觉得是好事。现在会追问：**"真实成本去哪了？谁在替它买单？"** 这一问，很多"降价惠民"就露出了隐性代价。

## 检验

- ✅ 能区分"价格"（你付的数）和"成本"（真实耗费）
- ✅ 能解释：为什么医保限价不等于医疗成本变低
- ✅ 看到"政府降价"时能追问"真实成本转移到哪了"
