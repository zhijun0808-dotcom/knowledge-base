# 外援激励扭曲——给政府非企业、助腐败/白象工程、无市场底线

## 核心观点
**"援助"预设结果（帮发展），实则：给政府钱→鼓励重建破产国企/建宫殿广场（观赏非生产）→无经济底线约束。**

对比马歇尔计划：
- 西欧：战后重建物质资本、已有工业技能/生活方式 → 成功
- 第三世界：缺人力资本（技能/制度），需几个世纪建立 → 援助更难奏效

扭曲机制：
1. **给政府非企业** → 无利润/亏损信号 → 资源配置随政治非经济
2. **国际机构（IMF/世行）分配巨资** → 官员对受援国政府巨大影响 → 附加条件不管成败
3. **无经济底线** → 豁免市场竞争淘汰 → 破产国企永生、白象工程长存
4. **贷款免除/展期/以新贷还旧贷** → 变相送礼 → 无偿还压力

鲍尔（发展经济学家）："官方援助更可能阻碍而非促进发展"。
私人渠道更大：美对穷国官方援助$310亿（2010），私人慈善$390亿、私人资本$1080亿、汇款$1000亿。**官方援助激励最小。**

## 出处
- 来源：《经济学的思维方式》第22章
- 页码：第22章"外国援助"节

## 思考
援助是**政治工具而非经济工具**——给政府、附政治条件、无破产约束。与[[帝国主义非财富转移主渠道——殖民地投资少、战败国失殖民地反繁荣]]、[[国际投资流向富非穷——风险规避胜过回报率]]同根：资源配置靠激励非善意。
