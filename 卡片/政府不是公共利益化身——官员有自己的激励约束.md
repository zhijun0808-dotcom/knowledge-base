# 政府不是公共利益化身——官员有自己的激励约束

## 核心观点
**政府不是一个统一的"公共利益化身"，而是由无数有自己利益、激励、议程的个人和机构组成。** 它们更多回应自身激励，而非抽象的"公共利益"。

证据：
- 美国联邦政府同时补贴烟草种植、限制吸烟
- 参议员莫伊尼汉："美国联邦政府像交战中的君主国"
- 印度政治家不敢私有化亏损国企（怕失工会票）、不敢取消电力/化肥/水补贴（怕失农民票）、不敢开除骚扰企业的巡视员（怕失公务员票） → 财政赤字失控
- 巴西公务员50岁退休拿全薪、遗孀继承、子女再继承 → 养老金入不敷出

**关键洞察**：私营企业为生存必须迅速认错纠错（亏损→破产）；政府犯错成本由纳税人承担，承认错误的政治代价由官员独自承担 → **政府不愿认错、不愿止损是理性的**。
