# 比较优势——谁的机会成本最低，谁就该做这件事

## 核心观点

当有多个人/团队/国家都能做同一件事时，**机会成本最低的那个人最应该做**。

比较优势不是看"谁更厉害"（那是绝对优势），而是看"谁放弃的最少"。

---

## 绝对优势 vs 比较优势

| | 绝对优势（直觉） | 比较优势（经济学） |
|--|----------------|-----------------|
| 看什么 | **谁做得好** | **谁放弃得少** |
| 问题 | 你什么都能做，为什么不全包？ | 两个人合作，各做哪件总产出最高？ |
| 举例 | 我写PPT 9分，同事6分 → 该我写 | 我写PPT放弃做数据表的机会成本很高 → 该让同事写 |

---

## 比较优势 = 机会成本在"分工合作"场景的运用

```
你已掌握的机会成本
    ↓
比较优势 = 比的是：你做这件事的机会成本 vs 他做这件事的机会成本
    ↓
谁的机会成本更低 = 谁做这件事的相对损失最小 = 就该谁做
```

| 你的现有卡片 | 回答的问题 | 比较优势的位置 |
|------------|-----------|-------------|
| 机会成本 = 决策的本质 | "选A放弃了什么？" | 机会成本的**延伸应用** |
| 稀缺→边际思维 | "资源有限，怎么选？" | 资源有限→不能样样自己来→需要分工 |
| 边际思维 MB vs MC | "一个人做多少？" | **对比对象不一样**：自己 vs 多人 |

## 四个例子

### ① 工作中的分工

你和同事都要做两件事：写周报和做PPT汇报。

| | 写周报 | 做PPT |
|--|-------|------|
| 你 | 8分 | 9分 |
| 同事 | 6分 | 2分 |

| 绝对优势思维 | 比较优势思维 |
|------------|------------|
| "你两样都强，全包了吧" | "你写PPT的优势最大（9分，同事才2分）" |
| 结果：你累死，同事闲着 | "同事写周报虽然差，但他做PPT更差" |
| | **→ 你做PPT，同事写周报，交换后总产出最大化** |

### ② 外包还是自己做（个人场景）

你是一个自由职业设计师，时薪300元。

| 选项 | 成本 |
|------|------|
| 自己做饭1小时 | 材料30元 |
| 点外卖 | 40元 |

| 直觉 | 比较优势 |
|-----|---------|
| "自己做饭省10块" | "自己做饭1小时的机会成本=300元设计收入" |
| | "点外卖只花40元 → 实际赚了260元" |

**不是懒——是你的时间用在别处价值更高。**

### ③ 家庭分工

| | 做饭 | 洗碗 |
|--|-----|------|
| 你 | 8分 | 6分 |
| 伴侣 | 9分 | 8分 |

你样样差，但：
- 伴侣的优势在做饭（差距最大）
- 你的"相对优势"其实是洗碗（你做这个损失最小）

**→ 伴侣做饭，你洗碗——总产出最高。**

### ④ 国际贸易（经典场景）

| 国家 | 生产1吨粮需工人数 | 生产1匹布需工人数 |
|-----|----------------|-----------------|
| A国 | 1人 | 2人 |
| B国 | 3人 | 4人 |

A国什么都强。但：
- A国的机会成本最低的是**粮食**（1人 vs B国3人，差距最大）
- B国的机会成本最低的是**布**（4人 vs A国2人，差距最小）
- **→ A国专业种粮，B国专业织布，交换后两国都更富**

## 与边际思维的分界线

| | 边际思维（MB vs MC） | 比较优势 |
|--|--------------------|---------|
| **决策对象** | 一个人 | 多个人/国家 |
| **比较什么** | 做N+1次的收益 vs 成本 | 你做这件事的代价 vs 他做这件事的代价 |
| **决策内容** | **做多少？** | **谁来做？** |
| **底层概念** | 边际收益、边际成本 | 机会成本 |

**两者不冲突，是互补：先确定"谁来做"（比较优势），再确定"做多少"（边际思维）。**

## 思考

以前总觉着"谁厉害谁上"——但比较优势告诉我，厉害的人也有机会成本，什么都自己干反而是浪费。**合作不是因为有人不行，是因为每个人都应该有自己相对最该做的事。**
