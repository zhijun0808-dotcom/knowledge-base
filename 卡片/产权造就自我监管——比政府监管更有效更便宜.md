# 产权造就自我监管——比政府监管更有效更便宜

## 核心观点
**产权的核心经济价值不是保护富人，而是造就"自我监管"——所有者因承担损失/获得收益，自动监管自己的资源，无需第三方监督。**

苏联 vs 美国农民对比：
- 苏联：农产品归"人民"，运输腐烂、包装马虎、无人在意 → 损失由全社会分担，运输者拿固定工资与保管好坏无关
- 美国：桃子是农民私有财产 → 自动挑出坏桃子防止腐烂扩散 → 不愿承担不必要亏损

**监管成本对比**：
- 政府监管：需雇佣监督者、监督监督者（谁监督监督者？）、苏联共产党蜂窝结构也无法根治腐败无效
- 产权自我监管：零额外成本，所有者利益与资源保护天然对齐

**产权看似保护富人，实则最惠及穷人**：
- 富人总钱多，但非富人群体总购买力更大 → 自由市场上豪宅地被开发商买走建平民公寓
- 租金管制/限制产权 → 诚实房东亏损退出、不诚实房东靠贿赂/偷工减料获利 → 房产从诚实者转移到不诚实者手中
- 香港租金管制后：房东半夜偷偷破坏设施逼租客走、纵火烧房改建商业 → 反而减少廉租房

**濒危动物都是"无主之物"**：山德士上校不会让鸡灭绝、麦当劳不会让牛灭绝；公地放羊→草场变荒漠，私有地旁边草长得好。

## 思考
产权常被攻击为"富人特权"，索维尔用实证反驳：**没有产权的国家，穷人连食物都买不起**。产权保护的是"资源不被浪费"的激励机制，受益的是全社会。
