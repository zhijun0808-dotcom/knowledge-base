# 通货紧缩伤害债务人——实际债务负担上升导致违约潮

## 核心观点
**通货紧缩（物价下降）看似让钱更值钱，实则毁灭债务人、拖垮银行、引发大萧条。**

机制：货币供给缩减 → 物价下降 → **名义债务固定，实际偿债负担上升**。

- 农民卖小麦价格从$1/蒲式耳跌到$0.50，但抵押贷款月供不变 → 实际还债成本翻倍
- 1896年布莱安竞选总统喊"别把人类钉死在黄金十字架上"，背景就是长期通缩压垮农民
- 1929-1933美国货币供给缩减1/3，物价下降但**合同固定的债务（房贷、企业借款）实际负担激增** → 9000+家银行倒闭、大规模止赎、失业率25%

**工资粘性加剧灾难**：胡佛/罗斯福坚持"不降工资维持购买力"，结果企业付不起高实际工资 → 大规模裁员。熊彼特直言：拒绝降薪让大萧条雪上加霜。

通缩还会让人**囤积现金、推迟消费/投资**（钱越存越值），总需求进一步萎缩 → 通缩螺旋。

反面教训：19世纪英国工业革命期产出激增、黄金供给跟不上 → 自然通缩，但没引发大萧条，因为**没有刚性债务合同+工资灵活**。

## 出处
- 来源：《经济学的思维方式》第17章
- 页码：第17章"通货紧缩"节

## 思考
通胀是"隐性税收"向持币者征收，通缩是"隐性违约"向债务人征收。**货币供给既不能太多（通胀），也不能太少（通缩）**，核心难点在于：债务合同以名义货币计价，而经济运行在实际购买力中。

这也是为什么现代央行设定**2%通胀目标**——留出缓冲，避免掉入通缩陷阱。
