# 现值是跨期价格信号：把未来收益折现到今天，迫使经济决策面对长远约束

## 核心观点

**现值 = 未来全部收益/损失的折现之和。** 它是一座桥，把未来的收益和成本拉到今天来定价。

因为现值存在，今天的资产价格**已经包含了未来的信息**——种果树的老人的地价、市政厅旁边建污水厂后的房价、石油公司钻探决策前的油价，都已经被现值重新定价了。

> **"资产的现值实际上就是对未来收益预期的加总，以及因为延期而产生的折现。"**（原文）

---

## 机制：未来如何影响今天的价格

### 正向：好维护带来今天的溢价

你给房子装了更耐用的管道和木料，它现在看起来和邻居的房子一样，但**市场价值已经更高了**——因为未来的维护成本更低，这部分节省被"折现"到了今天的价格里。

### 负向：坏消息立刻打折

市政府宣布明年在你家旁边建污水处理厂 → 房屋价值**立刻下降**，尽管污水厂还没动工。

> **任何能在预期上提高或降低未来收益或损失的东西，都会对今天所售资产的价格产生影响。**

### 最极端的例子：91 岁老人种果树

> 一位 91 岁的老人种植需要 20 年才能够成熟的果树也可能是合理的，因为土地的价值会由于这些果树立即提高。他可以在一个月后将这片土地卖掉，然后去巴哈马群岛生活——他现在的财富价值中就包括了还没有长出来的水果的价值，**未来出生的孩子可以吃上这些水果。**

**现值让"利在千秋"的决策在今天就被奖励。**

## 政治 vs 市场：谁受现值约束

| 市场（企业、债券、资产） | 政治（民选官员） |
|-------------------------|-----------------|
| 现值迫使今天面对未来的后果 | **不存在现值约束** |
| 标准普尔在加州电力危机时下调其债券评级——虽然加州当时有盈余，但未来要承担高税收高债务 | 政治家只需要撑到下一届选举 |
| 债券价格立刻反映长期风险 | 长期破坏性后果可以在大选后才显现 |

> **"经济与政治最大的不同之处在于，政治家不用关注下一届选举之后的未来后果。"**（原文）

这就是为什么金融市场比政治系统更擅长处理长期问题——不是因为金融家更聪明，而是因为**现值机制让长期后果必须在今天被定价**，而政治没有这个机制。

## 资源不会"耗尽"：现值的视角

这是一个经典的反直觉论证——用现值视角看"资源耗尽"的恐慌。

### "石油只够用13年"——然后呢？

1960 年畅销书预测：美国石油只够用 13 年。13 年后，已知储量反而更多了。

原因是：**已知储量不等于地底全部石油。它只是在当前价格和技术下值得开采的那部分。**

```
当前价格下，边际勘探成本 > 每桶石油的现值
  → 不值得继续勘探
    → "已知储量"停止增长
      → 专家："要耗尽了！"
        → 石油实际用掉 → 价格上升
          → 每桶现值上升
            → 勘探重新变得有利可图
              → 新油田被发现 → "已知储量"又增加了
```

### Simon-Ehrlich 赌局

经济学家西蒙打赌：任何人选五种自然资源，十年后实际成本都会下降。环保人士埃尔利希选了五种，**十年后全部降价。**

如果真的要耗尽了，它们的价格应该上升——因为更稀缺的资源的现值会上升。但现实是技术进步使获取成本持续下降。

### 2/3 的石油留在地下

> **"近 2/3 的石油仍然保留在地下油田中，因为在目前的科技和油价水平下，想要开采出油田中全部的石油——甚至只开采出大部分石油——成本太高了。"**（原文）

这不是"快用完了"，而是**在今天的现值下不值得全部挖出来。** 等油价涨到一定程度，它们自然会变成"可开采的储量"。

## 思考

这个概念的冲击力来自两个方向：

1. **对内**：现值解释了为什么"已知储量"和"总储量"是两回事——这是我们之前做 [[教育不一定等于人力资本，有市场价值的技能才是]] 时一样的陷阱：把**当前可测量的**当成**全部存在的**。

2. **对外**：政治没有现值约束——这解释了为什么长期问题（养老金、国债、气候变化）在民主政体里永远被拖延。不是政客坏，是**制度没有给"未来"定价的机制。**

91 岁老人种果树这个故事太漂亮了。它完美展示了：**市场让"为自己赚钱"和"为后代留下东西"是同一件事。**
