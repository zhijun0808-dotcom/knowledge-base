# 政府干预信贷分配——社区再投资法案酿成次贷危机

## 核心观点
**政治家用信贷政策追求社会目标，代价是金融体系脆弱化。**

1977年《社区再投资法案》（CRA）要求银行向低收入社区放贷，满足政府配额。90年代运动复兴，施压银行：
- 降低贷款标准（低信用分、零首付、收入不验证）
- 达不到配额 → 监管惩罚、合并申请被否

结果：
- 银行大量发放次级贷款 → 打包证券化卖给华尔街
- 房价上涨掩盖风险 → 2006年见顶回落 → 违约潮 → 银行倒闭、华尔街崩盘、纳税人救助

**政府干预信贷的通病**：
- 印度：70%存款贷给政府/国企（低回报），私企融资难
- 中国：国有银行90%贷给国企（亏损也贷），民企挤出
- 美国：CRA强迫银行贷给不合格借款人 → 次贷危机

**核心机制**：私人银行以盈利为目标，风险自担 → 贷给回报最高/风险最低处。政府强迫贷给"政治优先但经济不合格"对象 → **风险社会化、利润私有化（泡沫期） → 损失全民买单（破裂期）**。

## 思考
这张卡连接了三个层面：
1. **微观激励**：银行被迫降标准 → 单笔贷款风险上升
2. **中观传染**：证券化把局部风险传遍全系统
3. **宏观后果**：信贷扩张推高资产价格 → 崩盘引发衰退

与[[存款保险解决挤兑但引入道德风险]]同根：**政府介入金融风险分配，必然扭曲市场纪律**。区别是CRA是**显性指令**，存款保险是**隐性激励**。
