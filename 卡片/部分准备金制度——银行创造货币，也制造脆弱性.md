# 部分准备金制度——银行创造货币，也制造脆弱性

## 核心观点
**银行不是货币仓库，是货币制造机。** 部分准备金制度让银行只留一小部分存款作准备金，其余贷出去 → 贷款变成别人存款 → 再贷出 → **货币供给倍增**。

起源：金匠储藏黄金 → 发收据 → 发现收据当货币流通、黄金很少被全额取走 → 借出部分黄金收利息 → 现代银行雏形。

运作：
- 存款人存$100，银行留$10准备金，贷$90
- 借款人花$90 → 收款人存入银行 → 银行留$9，贷$81
- 循环下去，$100基础货币可衍生$1000存款货币（准备金率10%时）

**银行系统创造的信贷货币，远超政府发行的现金。** 2012年美银行资产$14万亿，远超流通现金。

**代价是脆弱性**：
- 存款人随时可取，贷款却长期（30年房贷、厂房抵押）**期限错配**
- 挤兑时资产无法快速变现 → 银行倒闭
- 多家银行同时挤兑 → 信贷蒸发 → 总需求骤降 → 大萧条（1930s美9000+银行倒闭，货币信贷缩减1/3）

## 出处
- 来源：《经济学的思维方式》第17章

## 思考
部分准备金是**效率与稳定的权衡**：
- 效率：释放闲置资金、支持大规模生产、降低融资成本
- 脆弱：依赖"存款人不同时取款"的信心游戏

美联储成立初衷是防挤兑，结果1930s却制造了史上最严重挤兑——**政府机构也会因激励扭曲、知识局限而失灵**。
