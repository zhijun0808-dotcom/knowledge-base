# 可靠的法律框架比公正的法律更重要

## 核心观点
**法律的"可靠性"（可预测、不随意变）比"公正性"对经济繁荣更关键。**

- 19世纪英国成为首个工业国，核心因素之一是**法律可靠**——投资者不用担心合同被政治随意改变、收入被没收、被索贿
- 东南亚殖民地引入欧洲法律替代部落统治 → 统一可靠框架 → 吸引华人/印度移民带来技术与企业家精神 → 经济转型
- 奥斯曼帝国基督徒/犹太人权利不如穆斯林，**但权利可靠** → 他们在商业/银行/工业超越穆斯林多数
- 甚至**歧视性但可靠**的法律，也比**公正但随意变动**的法律更利经济

反面：刚果资源丰富却只剩贫穷——政府无效、专断、腐败；玻利维亚警察贩毒偷车、官员住豪宅；埃及权贵杀人判死刑却让人"又吃惊又高兴"（因通常不受法律约束）。

**可靠性 = 防止他人干扰 + 防止罪犯/歹徒制造风险**。高犯罪率地区企业不敢落户、效率低、价格高（安保成本转嫁消费者）。

## 出处
- 来源：《经济学的思维方式》第18章

## 思考
这解释了为什么一些"不民主"但法治稳定的地区（如殖民地时期香港、新加坡早期）能繁荣，而一些"民主"但法律随意变动的国家却衰退。**投资者要的不是完美正义，而是可预期的规则。**

与[[政府不是公共利益化身——官员有自己的激励约束]]呼应：政府最大的经济贡献往往不是"做什么"，而是"承诺不做什么"（不随意改规则、不没收、不腐败）。
