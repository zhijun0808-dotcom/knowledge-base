# 企业财务体检单：现金流 + 资产结构核心指标

## 核心观点

看企业健康，先看血（现金流），再看骨（资产结构），最后看联合指标。

---

## 必看 5 个核心指标

| 指标 | 公式 | 健康标准 | 核心含义 |
|------|------|---------|---------|
| **利润含金量** | 经营现金流 / 净利润 | **> 1.2** | 利润里有多少水分 |
| **真正造血能力** | 自由现金流 FCF | **> 0 且增长** | 真正留给股东/债权人的钱 |
| **骨架造血效率** | 经营现金流 / 总资产 | **> 10%（优秀 > 15%）** | 每块骨头能生多少血 |
| **杠杆风险** | 资产负债率 | **< 60%（行业不同）** | 骨架里借来的骨头多不多 |
| **骨架转动效率** | 资产周转率 | **> 1（制造业 > 0.8）** | 一块骨头创造多少营收 |

---

## 红旗预警（任一触发 = 深挖）

| 预警信号 | 含义 |
|---------|------|
| 🚩 经营现金流长期 < 净利润 | 利润含水分大，可能粉饰 |
| 🚩 经营现金流长期 < 资本性支出 | 自己造的血不够养骨架，靠借钱/卖资产续命 |
| 🚩 存货/应收账款周转天数持续上升 | 骨架里长赘肉，货卖不出去/钱收不回来 |
| 🚩 短期借款激增、流动比率 < 1 | 短期偿债风险大，血液循环可能堵塞 |
| 🚩 商誉/无形资产占总资产 > 30% | 骨架里注水严重，一旦减值巨亏 |

---

## 进阶联合指标（深度体检）

| 指标 | 公式 | 优秀标准 | 含义 |
|------|------|---------|------|
| **ROIC** | NOPAT / 投入资本 | **> WACC** | 骨架产生的回报是否超过借钱成本 |
| **FCF / 总资产** | FCF / 总资产 | **> 10%** | 扣除维护骨架后，真正自由的血 |
| **现金流覆盖投资比** | 经营现金流 / 资本性支出 | **> 1.2** | 自己造的血够不够养骨架、长骨架 |
| **经营现金流 / 总负债** | 经营现金流 / 总负债 | **> 50%** | 用几年赚的钱能还清债 |
