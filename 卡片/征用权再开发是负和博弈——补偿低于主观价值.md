# 征用权"再开发"是负和博弈——补偿低于主观价值

## 核心观点
**政府以"公共利益"名义征用私产转给商场/酒店/游乐园，实为负和博弈：新业主获益<原主损失，社会净损失。**

机制：
1. **补偿=市场价<主观价值**：原主不愿卖才被强征，市场价低于其留用价值
2. **宣布即贬值**：官员公开讨论"重建"→潜在买家退缩→资产现值下跌→征用时按贬值后价补偿
3. **零和变负和**：新业主因征用权低价得地，获益来自原主损失；若补偿=全市场价，仍负和（原主不愿卖说明主观价值>市场价）
4. **政治包装**：官员称被征用者"自私阻碍进步"，照片对比"旧社区vs新商场"制造进步假象

案例：
- 2005凯洛诉新伦敦案：最高法院支持为"经济发展"征用私宅建商业项目
- 加州渡轮补贴：往返成本$108，票价$14，补贴$94/趟，年$3000万——纳税人买单
- 纽约地铁5分票价神圣不可侵犯→私企破产→市政府接手亏损→税收补贴

## 思考
这与[[价格管制不会消除稀缺，只是换了种方式配给]]、[[评判政策看它创造的激励，不看它宣称的目标]]同根：政治权力替代市场定价，资源流向政治连接者而非最高价值用途。征用权把"公共用途"(路/桥)无限扩张为"公共目的"(增税基/就业/游乐园)，产权保护名存实亡。
