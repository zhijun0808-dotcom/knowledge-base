# 大萧条是政府干预酿成的——股市崩盘不等于萧条

## 核心观点
**1929年股市崩盘后失业率峰值9%→6.3%自然下降；政府干预后失业率飙升至20%+并持续十年。**

时间线铁证：
- 1929年10月崩盘 → 2个月后失业9% → 1930年6月降至6.3% → **12个月内未破10%**
- 1930年6月《斯穆特-霍利关税法》签署（千名经济学家联名反对） → 5个月内失业率反弹破10% → 此后十年**从未跌破两位数**
- 胡佛/罗斯福前所未有干预：关税、工资管制、农价支持、尼克松式价格管制 → 每次干预后情况恶化

反面教材——1921年萧条：哈丁总统**只按税收下降削减开支**，什么都没做 → 失业11.7%→6.7%→2.4%，三年自愈
1987年股灾：里根**拒绝干预** → 媒体愤怒 → 经济自愈 → 20年"令人羡慕的稳定增长低通胀"

## 出处
- 来源：《经济学的思维方式》第20章
- 页码：第20章"大萧条"节

## 思考
这是全书最震撼的实证反驳："政府必须有所作为"是政治公理，非经济结论。**股市崩盘是症状，政府干预才是病因**。与[[评判政策看它创造的激励，不看它宣称的目标]]、[[政治激励与经济激励的错位——连任 vs 生存]]、[[政府项目不认错不止损——协和飞机、英法海底隧道]]同根。
