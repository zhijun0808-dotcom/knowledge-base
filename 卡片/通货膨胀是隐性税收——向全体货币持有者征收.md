# 通货膨胀是隐性税收——向全体货币持有者征收

## 核心观点
**通货膨胀本质上是一种税收——政府通过印钞，把全体货币持有者的购买力悄悄转移到自己手里。**

为什么政府爱用通胀而不加税？
1. **政治安全**：加税会引发选民反抗、甚至武装叛乱（如美国独立战争）；通胀隐蔽、缓慢，没人会为"物价上涨"上街革命
2. **覆盖面广**：所得税只对"富人"或纳税人，通胀向**所有持有货币的人**征收——从亿万富翁到靠养老金过日子的老人
3. **无需立法**：转动印钞机即可，绕过议会辩论

**富人能规避，穷人不能**：富人把钱投股票、房产、实物资产，随通胀升值；低收入群体工资调整滞后、存款多为现金/定期，**实际购买力被无声切走**。

历史证据：
- 亚历山大大帝掠夺波斯金银 → 希腊物价飞涨
- 西班牙从美洲运回黄金 → 全欧物价上涨
- 美国内战/一战/二战 → 政府印钞买军火，储蓄贬值
- 1920s德国恶性通胀：1920年40马克=1美元 → 1923年11月4万亿马克=1美元，毕生积蓄买不包烟
- 1990s拉美年通胀600%，实际产出下降；通胀平缓后年增6%

凯恩斯警告：**"持续通胀让政府秘密、不被察觉地没收公民大量财富。"**

## 思考
通胀税最阴毒之处：**名义上不加税，实质上全民买单**。政治家说"不加税"时往往在印钞。看政策别听承诺，看货币供应量增速。
