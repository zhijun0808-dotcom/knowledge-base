# 最低工资法本意护穷人，实则把低生产率者的就业权立法剥夺了

## 核心观点

最低工资法的宣称目标是"保护低收入者"，但实际后果是：**把边际生产价值低于法定工资的人，直接"立法"挡在就业大门外——真正的最低工资是零（失业）。**

---

## 核心机制：价格下限 → 劳动力过剩（失业）

最低工资法 = **劳动力市场的价格下限**。

| 正常市场 | 加上最低工资法 |
|---------|--------------|
| 工资 = 边际生产价值 | 法律强行把工资托高到 $X |
| 谁都能以各自生产率对应的工资就业 | 边际生产价值 < $X 的人 → **雇不起** |
| 谁都能就业（工资高低不同） | 低生产率者 → **被"定价"出局** |

> **真正的最低工资永远是零**（失业）。

---

## 谁最惨？**年轻、低技能、少数族裔**

| 群体 | 为什么最惨 |
|------|-----------|
| **年轻人/新人** | 没经验、边际生产价值低 → 最先被"定价"出局 |
| **低技能者** | 生产率低 → 最先被"定价"出局 |
| **少数族裔/边缘群体** | 往往技能少、歧视多 → 最先被挤出 |

> **索维尔铁律数据**：1948年美国黑白青年失业率相当；最低工资连续上调后，**黑人青年失业率飙升、白人青年基本不动**。

---

## 谁在推？谁得利？

| 推手 | 真实动机 |
|------|---------|
| **工会** | 会员工资远高于最低工资 → 推高地板 = **挤掉低价竞争对手** = 保护会员饭碗 |
| **政客** | "帮穷人"的口号好拉选票；失业是"二阶后果"，选民不买单 |
| **大企业** | 能自动化/承担高工资 → 推高门槛 = **挤死小竞争对手** |

> **最低工资法本质上是"高生产率者联合立法，把低生产率者挤出市场"。**

---

## 反证：没最低工资的国家，失业率更低

| 国家 | 最低工资 | 失业率 |
|------|---------|--------|
| 瑞士 | 无 | ~2% |
| 新加坡 | 无 | ~2% |
| 美国（1930前） | 无 | 1.8% |

> **没有价格下限，劳动力市场自己找平衡——低生产率的人也能以低工资就业，积累经验、再涨工资。**

---

## 为什么这法还在？

| 原因 | 解释 |
|------|------|
| **意图好、后果坏** | 政客只看"宣称目标"（护穷人），不看"激励后果"（造失业） |
| **受益者有组织** | 工会、大企业有游说能力；受害者（年轻/低技能）无声无力 |
| **失业是"二阶后果"** | 选民看得见"工资涨了"，看不见"岗位没了" |

---

## 出处

- 托马斯·索维尔《经济学的思维方式》第 11 章——最低工资法
- 核心句："真正的最低工资永远是零"

## 思考

以前觉得"最低工资法保护穷人"。现在明白：**它把"低生产率者"的就业权立法剥夺了。** 真正帮穷人的不是定高工资，而是让他们能进职场、积累经验、提高生产率、再自然涨工资。
