# 比较优势——机会成本决定谁该生产什么

## 核心观点
**即使一个国家在所有产品上都有绝对优势（生产效率全面更高），它仍应专注于比较优势产品，通过贸易获益。**

核心逻辑：资源稀缺→生产一种=放弃另一种。比较优势看的不是"谁生产得更快"，而是"谁放弃得更少"。

经典案例——美加椅子/电视机：
- 美工人：月产500椅子 或 200电视机
- 加工人：月产450椅子 或 100电视机
- 美国两项都强（绝对优势），但**加拿大生产椅子的机会成本更低**（放弃100电视机换22.5万椅子，美国放弃200电视机换10万椅子）
- 专业化后：美产电视机、加产椅子 → 总产出：电视机10万→10万、椅子19万→22.5万，**总和增加**

生活类比：眼科医生洗车虽比雇工快，但医生时间机会成本=少做几台手术，**医生应雇人洗车、专心治眼**。

贫穷国家受益最大：无论多穷，总有某种产品相对效率最高，自由贸易给每国留出立足之地。

## 思考
这是国际贸易理论的皇冠明珠。大多数人直觉用"绝对优势"（谁更强），经济学用"比较优势"（谁放弃少）。与[[机会成本就是显性成本 + 隐性成本]]、[[稀缺让竞争不可避免，问题是竞争以什么形式发生]]同根同源——核心都是**稀缺资源的替代用途**。
