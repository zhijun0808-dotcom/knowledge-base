# 补贴"负担得起"实则全社会付全成本——价格低于成本

## 核心观点
**政府补贴让产品"负担得起"，不降低真实成本，只是把账单分摊给全体纳税人——最终人们为只值$6的东西付$10（$4税+$6价）。**

机制：
- 产品成本$10，大多数人只愿付$6 → 政治定义为"问题"
- 补贴$4 → 售价$6，购买者付$6，纳税人付$4 → **全社会实付$10**
- 补贴管理成本>0 → 总成本>$10，资源误配更甚
- 政治现实：价格常压到$6以下，成本却>$10（照顾那一半付不起$6的人）

对比：**直接给穷人现金/代金券**，让其他人按市场价买 → 效率高得多，目标精准。

经典扭曲：
- 国家公园：老人$10终身票，其他人$25/次 → 老人多获益，但关键是老人投票率高
- 城市高尔夫场、交响乐团：穷人极少用，却全民补贴
- 渡轮补贴：$14票价，成本$108，补贴$94/趟
