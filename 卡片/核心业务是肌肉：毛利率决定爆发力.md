# 核心业务是肌肉：毛利率决定爆发力

## 核心观点

核心业务是企业的肌肉，**毛利率 = 肌肉的质量和爆发力**。

| 肌肉指标     | 企业对应           | 含义             |
| -------- | -------------- | -------------- |
| **肌肉量**  | **毛利率**        | 单位营收能留下多少钱养活自己 |
| **肌肉密度** | **毛利率稳定性**     | 穿越周期不崩，说明有真护城河 |
| **爆发力**  | **毛利率 × 营收增速** | 既有厚度又有速度，才能复利  |

---

## 核心判断标准

| 毛利率水平 | 肌肉状态 | 典型企业 |
|-----------|---------|---------|
| **> 70%** | **钢铁侠** | 微软、Adobe、茅台、贵州茅台 |
| **40%-70%** | **健美运动员** | 苹果、可口可乐、海天味业 |
| **20%-40%** | **普通健身者** | 大多数制造业、零售业 |
| **< 20%** | **肌肉萎缩** | 传统制造、贸易型、价格战红海 |

> **核心洞察**：毛利率低 ≠ 不能活，但**没有定价权**，抗风险极弱，稍微涨原材料/降价就亏损。

---

## 核心拆解：毛利率 = 定价权 - 成本控制力

```
毛利率 = (售价 - 单位成本) / 售价
```

| 维度 | 强肌肉表现 | 弱肌肉表现 |
|------|-----------|-----------|
| **定价权** | 有品牌溢价、转嫁成本、客户不敏感 | 价格战、拼低价、客户极度敏感 |
| **成本控制** | 规模效应、供应链整合、技术降本 | 被上游卡脖子、规模不够分摊固定成本 |

---

## 进阶看法：毛利率 ≠ 净利率

| 指标 | 看什么 | 陷阱 |
|------|-------|------|
| **毛利率** | **核心业务肌肉质量** | 高毛利但高销售费用 = 肌肉好但穿厚棉袄跑不动 |
| **净利率** | 全身协调后的最终战果 | 可能靠投资收益/政府补贴/卖资产撑起来 |

> **核心判断**：先看毛利率（肌肉本身），再看费用率（穿多厚衣服），最后看净利率（最终战果）。

---

## 红旗预警

| 信号 | 含义 |
|------|------|
| 🚩 毛利率连续 3 年下滑 > 3% | 护城河在塌，肌肉在流失 |
| 🚩 毛利率高但销售费用率 > 30% | 肌肉好但穿太厚衣服跑不动 |
| 🚩 毛利率 < 行业平均 10% 以上 | 核心竞争力缺失，可能陷入价格战 |
| 🚩 新产品毛利率显著低于老产品 | 新业务没肌肉，靠老业务输血 |

---

## 关联指标组合拳

| 组合 | 含义 |
|------|------|
| **高毛利 + 高周转** | 肌肉既强又快（如可口可乐、海天味业） |
| **高毛利 + 低周转** | 肌肉强但笨重（如高端白酒、奢侈品） |
| **低毛利 + 高周转** | 肌肉薄但跑得快（如Costco、大润发） |
| **低毛利 + 低周转** | **肌肉萎缩 + 行动迟缓 = 待淘汰** |
