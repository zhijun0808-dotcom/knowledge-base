# 时间就是金钱——延迟不是零成本，是把利息成本强加给别人

## 核心观点
"时间就是金钱"不止是一句谚语，它是一条深刻的经济学原理：**谁有能力拖延，谁就能把成本强加给另一方。** 而且这些成本往往是巨额的。

经典场景：建房开发商借款数百万美元买地、建设，每天都要付利息。如果因为环境评估、政府审批、法律质疑等原因延迟一年，新增的利息成本远超评估报告本身的直接成本。**延迟造成的利息支出，往往比被争论的那个议题本身大得多。**

这意味着：
1. **官僚拖延不只是"效率低"，而是有真实的经济成本**——由被拖延方承担
2. **环保审查、卫生条例、规划许可等制度**，即使出于正当目的，也能被用作"以拖延增加成本"的武器，阻止反对者不喜欢的经济活动
3. **贿赂的经济逻辑**：在腐败严重的国家，行贿加快流程是"用一笔可控的钱换一笔不可控的利息成本"
4. **提高退休年龄本质上是对养老金受益人的违约**——把财政负担转嫁到下一代，违背了当初的合约

更一般地说：**时间成本是隐性的，所以它常常被政策讨论忽略。但忽略不意味着不存在，它只是意味着这笔成本由无法发声的人承担。**

## 思考
这句话触动我的是它揭示了一个**权力不对称**——"能力"在这里指的是"让对方没法快速继续的能力"。不管是官僚、环保组织、还是工会，谁有能力制造延迟，谁就有能力把成本强加出去。

与 Ch11 最低工资的逻辑形成对照：在 Ch11，看起来帮穷人的政策实际上伤害了穷人。这里也是——看起来保护环境的审查制度，实际上可能通过延迟成本减少了房屋供给，最终推高了房价，让需要住房的人付出更高代价。殊途同归。
