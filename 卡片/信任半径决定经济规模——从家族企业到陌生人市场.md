# 信任半径决定经济规模——从家族企业到陌生人市场

## 核心观点
**肯尼斯·阿罗所谓的"信任半径"——人们愿意与陌生人合作的范围，直接决定了经济能发展多大。**

信任半径层级：
1. **仅限家庭**：马尔加什粮商不敢雇工（怕偷窃）→ 企业规模受家族大小限制
2. **封闭群体内部**：印度马尔瓦尔人、纽约哈西德犹太人、东南亚华人 → 靠群体内部社会制裁（蒙羞、失去地位、切断关联）实现巨额口头交易，**交易成本极低**
3. **普遍陌生人社会**：丹麦丢钱包几乎全归还、日本/以色列外交官交罚单 → **普遍诚信**支撑大规模陌生人市场

**经济后果**：
- 低信任社会：企业靠家族、靠关系、靠贿赂官员 → 规模小、效率低、创新慢
- 高信任社会：陌生人能汇集海量资金（股市、养老基金）、分工极细、创新快

**政府的角色**：不能直接"制造诚信"，但能通过**可靠的法律框架、惩罚欺诈、保护产权**间接支撑信任半径扩大；反之，**租金管制、繁文缛节、腐败容忍**会压缩信任半径，迫使经济退回"熟人圈子"。

## 思考
这张卡解释了为什么**同样是市场经济，有的国家繁荣、有的停滞**——核心差异不在"有没有市场"，而在"市场能容纳多大范围的陌生人合作"。

与[[可靠的法律框架比公正的法律更重要]]、[[产权造就自我监管——比政府监管更有效更便宜]]形成"制度三角"：可靠法律 → 扩大信任半径 → 产权自我监管生效 → 经济规模跃升。
