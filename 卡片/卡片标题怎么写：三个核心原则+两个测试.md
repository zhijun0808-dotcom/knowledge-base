# 卡片标题怎么写：三个核心原则 + 两个测试

## 核心观点

卡片标题必须是**一句完整的话**，表达一个观点，而不是一个名词。

标题的作用是**指方向**，核心观点的作用是**填内容**。

---

## 三个核心原则

### 1. 一句话原则
```
✅ 输入就是输出，不需要为写什么发愁
❌ 输入输出

✅ 记录是把瞬间想法锚定为思想成果的关键一步
❌ 记录
```

### 2. 标题指方向，核心观点填内容
```
标题：各种效应现象的背后都藏着本质
核心观点：本质是人类大脑存在认知捷径，各种效应是这些捷径在特定场景下的表现形式

标题：元认知的分水岭：被动接收 vs 主动调控
核心观点：被动 = 遇到问题才启用，主动 = 没事也主动启用
```

### 3. 标题写完后，如果核心观点没东西写 → 追问标题里的关键词
```
标题：各种效应背后都藏着本质
→ 关键词"本质" → 追问"本质是什么？" → 核心观点自然出来
```

## 两个测试

| 测试 | 问自己 |
|------|-------|
| **10 秒测试** | 朋友 10 秒问你这张卡片说什么，你能直接念标题吗？ |
| **3 年测试** | 3 年后只看标题，还能想起这张卡片在说什么吗？ |

## 常见错误对照

| ❌ 错误写法  | ✅ 正确写法               | 为什么          |
| ------- | -------------------- | ------------ |
| 机会成本    | 机会成本就是显性成本 + 隐性成本 | 标题要是观点，不是名词  |
| 写作技巧    | 写作不是从零开始，而是从卡片盒拼装    | 标题要是观点，不是主题  |
| 记笔记的重要性 | 记录是把瞬间想法锚定为思想成果的关键一步 | 标题要是观点，不是重要性 |

---

## 快速生成标题的三个句式

| 句式 | 例子 |
|------|------|
| **……不在于……而在于……** | 写作不在于天赋，而在于系统 |
| **核心在于……** | 核心在于：输入和输出被割裂了 |
| **本质上……** | 本质上：卡片盒是把思考外包给系统 |

## 检验清单

写完标题后，对着这三个问题自测：

- [ ] **10 秒测试**：朋友 10 秒问你这张卡片说什么，你能直接念标题吗？
- [ ] **3 年测试**：3 年后只看标题，还能想起这张卡片在说什么吗？
- [ ] **费曼测试**：标题能用大白话讲给没读过的人听吗？
