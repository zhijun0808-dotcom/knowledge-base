# 价格管制不会消除稀缺，只是换了种方式配给

## 核心观点

价格管制的初衷往往是"让穷人买得起"，但结果恰恰相反——**人为压低或抬高价格，不会改变背后的稀缺事实，只会把配给方式从价格变成短缺排队、质量下降、黑市交易或资源浪费。**

就像把体温计放进冷水里——读数是降了，但烧没退。

## 价格天花板（最高限价）：典型例子——租金管制

```
政府把租金定在市场价以下
  → 租客觉得便宜 → 需求量暴增
  → 房东不愿维修（修了也涨不了租）→ 质量恶化
  → 开发商不愿盖新房（不赚钱）→ 供应萎缩
  → 房子更少、更破，真正需要的人反而租不到
```

**具体表现：**

| 结果       | 说明                           |
| -------- | ---------------------------- |
| **短缺**   | 想要的人 > 可租的房子                 |
| **质量下降** | 房东不维护，房屋加速老化                 |
| **囤积**   | 租到便宜房的人占着不走（一人住三居室），不释放给需要的人 |
| **黑市**   | 地下交易、"好处费"变相补回市价差            |

**数据佐证：** 曼哈顿实施租金管制的公寓中，46% 只住了 1 个人——全美平均仅 27%。管制反而让稀缺的住房被低效占用。

## 价格地板（最低限价）：典型例子——农业补贴

```
政府把农产品价格托在市场价以上
  → 农民觉得有利 → 过量生产
  → 消费者嫌贵 → 没人买
  → 政府掏钱买下剩余 → 浪费纳税人钱
  → 剩余产品倾销到国际市场 → 摧毁别国农民
```

| 结果 | 说明 |
|------|------|
| **过剩** | 卖不掉的粮食堆积 |
| **资源浪费** | 土地、人力、水被用来生产没人要的东西 |
| **质量停滞** | 反正好卖（给政府），何必改进 |
| **国际连锁** | 国内过剩出口压低全球价格，伤及他国生产者 |

## 稀缺 ≠ 短缺——最被忽视的区别

| 概念 | 含义 | 能否被消除 |
|------|------|----------|
| **稀缺**（scarcity） | 资源客观上不够所有人各取所需 | **不能**——物理现实，永恒存在 |
| **短缺**（shortage） | 在某个价格下，需求量 > 供应量 | **能**——让价格涨到均衡点就消失 |

**错误直觉：** "价格高 → 说明短缺 → 政府压价 → 问题解决"
**正确逻辑：** "价格高 → 说明稀缺 → 政府压价 → 信号消失 → 没人增产 → 短缺更严重"

**海滨房无论定什么价都是稀缺的**——因为海滨土地就那么多。价格只是把"谁住得上"的信息传出来。压价不会让房子多出来，只会让排队和走后门决定谁住。

## 跨越时空的一致性

从古罗马戴克里先大帝的价格令、到法国大革命巴黎的管制、到二战后斯德哥尔摩和纽约的租金管制——**结果一模一样：短缺、质量下降、黑市、供应萎缩。**

不是意识形态，是因果规律。正如索维尔引用的：

> **"租金管制是目前已知的摧毁一个城市最有效的手段——除了轰炸。"** ——瑞典经济学家 Assar Lindbeck

## 思考

以前以为"价格高=有问题=该管"，现在才理解：**高价格是在喊"不够了，快来增产"；压价等于把报警器关了，但火还在烧。**

租金管制听起来是在帮穷人，但它阻止了开发商盖新房、阻止了房东修旧房——最终受伤最深的恰恰是那些还没租到房的人（通常是更穷的人）。

**一项政策的好坏，要看它创造的激励，而不是看它宣称的希望。**
