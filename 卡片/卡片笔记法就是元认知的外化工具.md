# 标题：卡片笔记法就是元认知的外化工具

## 核心观点
<!-- 核心观点要说清楚为什么 -->
元认知 = 监控 + 调控过程。
卡片笔记法通过“用自己的话写”，“建链接” “建结构笔记” 定期回顾
把隐形的元认知过程外化成可操作、可迭代的外部系统。

核心对应
- 用自己的话写—监控理解程度
- 建链接 — 调控知识链接
- 建结构笔记 — 调控知识结构
- 定期回顾 — 监控遗忘/固化
