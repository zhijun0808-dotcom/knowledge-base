## 经济因果闭环（全书的骨架）

```
稀缺（永远存在）
  ↓
买卖双方各自用边际思维决策
  ├─ 需求方：边际效用递减
  └─ 供给方：边际成本递增
  ↓
两边对撞 → 价格（千万人边际决策的汇总信号）
  ↓
企业据价格算利润/亏损 → 兴衰（决策模式是否有效的信号）
  ↓
对的扩张（资源继续流入）/ 错的淘汰（资源释放重配）
  ↓
回到稀缺 → 循环永不停止
```

## 🔧 方法论：怎么分析市场（显微镜，不是标本）

> 上面的闭环是"标本"，下面这几张是"显微镜"——你拿什么逻辑去读市场现象。

- [[评判政策看它创造的激励，不看它宣称的目标]] ← 核心方法论：因果>意图，激励>目标
- [[稀缺让竞争不可避免，问题是竞争以什么形式发生]] ← 稀缺是舞台，价格竞争是最好的演法
- [[系统因果：经济里的因果是网状互馈，不是线性单链]] ← 别找单一凶手，找互馈回路
- [[价格≠成本：压低消费者支付的价格，不等于压低生产的真实成本]] ← 掐掉价格显示，成本仍在

## 🧰 边际思维：核心工具集

> 分析单位是"边际"而非"总量"——以下是一整套分析工具，从起点到汇总信号。

- [[稀缺资源迫使你不断用边际思维做决策]] ← 起点：稀缺是边际思维的驱动条件
- [[交易不创造东西，但创造价值：从不要的人流向要的人]] ← 前提：边际交换产生增益
- [[边际思维的两端：消费怎么用，生产怎么造]] ← 工具总览：需求方 MB vs 供给方 MC
- [[边际效用递减：第 N 个包子没第一个香]] ← 需求方工具：MB 随消费量递减
- [[边际收益递减——每多卖一个，赚得越少]] ← 供给方工具：MC 随产量递增
- [[增量收益（MB）= 每个决策者都有自己的价值标尺]] ← 微观基础：每个人的主观价值尺度不同
- [[机会成本就是显性成本 + 隐性成本]] ← 参照系：边际决策的真正成本是放弃的次优选择
- [[价格是千万人边际决策的汇总信号]] ← 汇总：千万人边际对撞的结果，协调全社会
- [[价格是供需对撞的信号，企业兴衰是供给端决策模式的信号]] ← 汇总的企业尺度：从价格到企业兴衰同源同逻辑

## 📌 政策误判案例

> 同一套工具，用于拆解政策最常见的认知误区——意图好≠结果好。

- [[价格管制不会消除稀缺，只是换了种方式配给]] ← 价格失灵的后果：管制不消灭稀缺，只转移成本
- [[最低工资法本意护穷人实则把低生产率者的就业权立法剥夺了]] ← 价格下限案例：最低工资制造劳动过剩
- [[自由市场里垄断活不长，长命垄断多半是政府养的]] ← 竞争切断的案例：真垄断的源头是政府权力
- [[反垄断法常帮倒忙：本想拆垄断，却保护低效]] ← 管制误判的案例：反垄断常保护低效而非破除垄断

## 🏭 要素市场定价

> 产品市场的定价逻辑（价格=供需对撞）同样适用于要素市场——只是每种要素各有特殊性。

- [[工资=边际生产价值，老板给不了也不用给]] ← 劳动要素的价格机制
- [[劳动不是普通商品：是人、技能不可转移、培训有外部性]] ← 劳动的特殊性：人不可交易，培训供给不足
- [[教育不一定等于人力资本，有市场价值的技能才是]] ← 人力资本的真实来源：实践技能 > 学历信号
- [[利率是资金的价格，人为压低它和价格管制一样扭曲配置]] ← 资金要素的价格，配置资本的跨期流动

## 🌍 扩展维度

> 边际思维的前提条件进一步打开后的推论。

- [[稀缺的是资源，更稀缺的是把资源变成财富的能力]] ← 稀缺的延伸：转化效率比资源量更重要
- [[比较优势——谁的机会成本最低，谁就该做这件事]] ← 分工逻辑：机会成本决定最优配置

## 金融市场：时间与风险的跨期配置

> 经济因果闭环讲"当下"的配置（价格→利润→兴衰）；金融市场层加上了**时间维度和风险维度**。

- [[时间不确定性和风险都有价格——利率投机库存现值是四种跨期配置机制]] ← 第13章总纲串联卡：四种机制都是价格，分别配置时间/风险/确定性/未来
- [[利率是资金的价格，人为压低它和价格管制一样扭曲配置]] ← 利率=时间的价格，资金市场的跨期配置信号
- [[投机不是赌博，是降低社会固有风险的专业化分工]] ← 风险转移的专业化：让不愿承担的人把风险转给愿意承担的人
- [[库存是知识不足时的对冲，过大过小都是错，最优由市场试错发现]] ← 知识的替代品：用实物缓冲不确定性
- [[现值是跨期价格信号：把未来收益折现到今天，迫使经济决策面对长远约束]] ← 跨期折现：把未来的约束拉回今天的决策

## 企业解剖对比人体的完整隐喻

> 把企业比作人体：每个财务指标对应一个器官系统。

- [[现金流≠利润，现金流是企业的血液]] ← 血液系统：存活的前提是现金流不断，不是利润为正
- [[现金流与资产结构：血液与骨架的共生关系]] ← 骨架系统：资产结构决定抗风险能力
- [[利润是社会为效率付的价，亏损是资源该挪窝的警报]] ← 神经系统/反馈：感知+纠错，驱动兴衰
- [[规模经济有拐点：企业不会无限大]] ← 运作原理：规模经济vs不经济，封顶企业规模
- [[中间商不是吸血，是价值链上更高效的一环]] ← 运作原理：中间商创造匹配价值
- [[企业的兴衰不是故障，是经济自我更新的机制]] ← 新陈代谢视角，由利润/亏损信号驱动
