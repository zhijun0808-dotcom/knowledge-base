## 经济因果闭环（全书的骨架）

```
稀缺（永远存在）
  ↓
买卖双方各自用边际思维决策
  ├─ 需求方：边际效用递减
  └─ 供给方：边际成本递增
  ↓
两边对撞 → 价格（千万人边际决策的汇总信号）
  ↓
企业据价格算利润/亏损 → 兴衰（决策模式是否有效的信号）
  ↓
对的扩张（资源继续流入）/ 错的淘汰（资源释放重配）
  ↓
回到稀缺 → 循环永不停止
```

## 🔧 方法论：怎么分析市场（显微镜，不是标本）

> 上面的闭环是"标本"，下面这几张是"显微镜"——你拿什么逻辑去读市场现象。

- [[评判政策看它创造的激励，不看它宣称的目标]] ← 核心方法论：因果>意图，激励>目标
- [[稀缺让竞争不可避免，问题是竞争以什么形式发生]] ← 稀缺是舞台，价格竞争是最好的演法
- [[系统因果：经济里的因果是网状互馈，不是线性单链]] ← 别找单一凶手，找互馈回路
- [[价格≠成本：压低消费者支付的价格，不等于压低生产的真实成本]] ← 掐掉价格显示，成本仍在

## 边际思维-经济学最核心的分析工具

- [[稀缺资源迫使你不断用边际思维做决策]]
- [[交易不创造东西，但创造价值：从不要的人流向要的人]]
- [[边际思维的两端：消费怎么用，生产怎么造]]
- [[边际效用递减：第 N 个包子没第一个香]]
- [[边际收益递减——每多卖一个，赚得越少]]
- [[价格是千万人边际决策的汇总信号]] ← 价格作为协调信号
- [[价格管制不会消除稀缺，只是换了种方式配给]] ← 人为压制价格的结果（反面验证）
- [[最低工资法本意护穷人实则把低生产率者的就业权立法剥夺了]] ← 同主题：价格下限→过剩（失业）
- [[自由市场里垄断活不长，长命垄断多半是政府养的]] ← 同主题：政府掐断竞争（牌照/关税养垄断）
- [[反垄断法常帮倒忙：本想拆垄断，却保护低效]] ← 同主题：管制意图好、后果坏的反面验证
- [[价格是供需对撞的信号，企业兴衰是供给端决策模式的信号]]
- [[工资=边际生产价值，老板给不了也不用给]] ← 要素价格的定价机制
- [[劳动不是普通商品：是人、技能不可转移、培训有外部性]] ← 劳动市场特殊问题：为什么劳动不是普通商品（人/不可转移/培训外部性）
- [[教育不一定等于人力资本，有市场价值的技能才是]] ← 人力资本≠正规教育：实践经验和有市场价值的技能才是真人力资本
- [[利率是资金的价格，人为压低它和价格管制一样扭曲配置]] ← 利率=跨期的价格信号；人为压低=资金市场的价格管制，扭曲配置
- [[增量收益（MB）= 每个决策者都有自己的价值标尺]]
- [[机会成本就是显性成本 + 隐性成本]]
- [[稀缺的是资源，更稀缺的是把资源变成财富的能力]]
- [[比较优势——谁的机会成本最低，谁就该做这件事]]

## 金融市场：时间与风险的跨期配置

> 经济因果闭环讲"当下"的配置（价格→利润→兴衰）；金融市场层加上了**时间维度和风险维度**。

- [[利率是资金的价格，人为压低它和价格管制一样扭曲配置]] ← 利率=时间的价格；资金市场的配置信号
- [[投机不是赌博，是降低社会固有风险的专业化分工]] ← 风险转移的专业化，和经济整体降低风险
- [[库存与现值——跨期决策的两种工具]] ← 库存=知识的替代品，现值=把未来收益折现到今天
- [[现值将未来带入现在，决定资源跨期配置]] ← 资产的当前价值=它未来全部收益的折现之和
