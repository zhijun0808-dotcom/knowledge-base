# 📌 15个观点句式工具箱

> 把现象压出观点，把模糊变成锋利。

---

## 用法

遇到一段叙述或一个现象，想不出写什么卡片时，从这个结构笔记里挑 2-3 个句式套进去试。

---

## 因果类

- → [[句式-核心在于——追问原因]]：**核心在于……**
- → [[句式-本质上——揭示本质]]：**本质上……**
- → [[句式-不在于而在于——指出误判]]：**不在于……而在于……**
- → [[句式-不是的问题而是的问题——归因于结构]]：**不是……的问题，而是……的问题**
- → [[句式-取决于——供需视角]]：**……取决于……**

## 反转类

- → [[句式-恰恰相反——反驳常识]]：**恰恰相反 / 实际上……**
- → [[句式-反过来想——逆向思考]]：**反过来想……**
- → [[句式-看起来实际上——识别陷阱]]：**看起来……实际上……**

## 推演类

- → [[句式-这意味着——推导后果]]：**这意味着…… / 长期来看……**
- → [[句式-如果那就——极限思维]]：**如果……那就……**

## 边界类

- → [[句式-不等于——区分概念]]：**……≠……**
- → [[句式-的前提是——指出前提]]：**……的前提是……**
- → [[句式-只适用于不适用于——条件限定]]：**……只适用于……不适用于……**

## 表达类

- → [[句式-就像一样——打比方]]：**……就像……一样**
- → [[句式-分为三个层次——程度分级]]：**……分为三个层次**

---

## 使用场景示例

| 场景 | 推荐句式 |
|------|---------|
| 看到一个现象，想挖深 | [[句式-核心在于——追问原因]] 或 [[句式-本质上——揭示本质]] |
| 想反驳一个常见观点 | [[句式-恰恰相反——反驳常识]] 或 [[句式-不在于而在于——指出误判]] |
| 想推演一个事实的结果 | [[句式-这意味着——推导后果]] 或 [[句式-如果那就——极限思维]] |
| 想区分两个易混概念 | [[句式-不等于——区分概念]] |
| 想让人印象深刻 | [[句式-就像一样——打比方]] 或 [[句式-分为三个层次——程度分级]] |
| 想指出常见误区 | [[句式-看起来实际上——识别陷阱]] 或 [[句式-反过来想——逆向思考]] |
